Diseña la interfaz gráfica de la aplicación con un estilo moderno, minimalista y tecnológico, inspirado en una estética limpia tipo startup SaaS.  
  
La interfaz debe usar una base visual clara, con fondo blanco azulado muy suave y elementos gráficos flotantes. El diseño debe sentirse ordenado, fresco, digital y confiable.  
  
### Paleta de colores  
  
- Fondo principal: #F3F9FF  
- Blanco puro para tarjetas y contenedores: #FFFFFF  
- Azul principal vibrante: #007BFF  
- Azul secundario: #1E90FF  
- Azul claro decorativo: #DCEEFF  
- Azul muy pálido: #EAF4FF  
- Verde suave de acento: #69E785  
- Amarillo / naranja de acento: #FFA51F  
- Rojo suave para alertas o estados: #FF5A5F  
- Texto principal oscuro: #1F2933  
- Texto secundario gris azulado: #6B7C93  
- Bordes suaves: #D8E6F3  
  
### Estilo general  
  
- Diseño minimalista con bastante espacio en blanco.  
- Tarjetas con bordes redondeados grandes.  
- Sombras muy suaves y difusas.  
- Elementos gráficos tipo ilustración vectorial.  
- Uso de formas geométricas: círculos, líneas, rectángulos redondeados, puntos y curvas.  
- Evitar interfaces pesadas o saturadas.  
- La app debe sentirse ligera, amigable y profesional.  
  
### Tipografía  
  
Usar una fuente sans-serif moderna como:  
- Inter  
- Poppins  
- Manrope  
- SF Pro Display  
  
Jerarquía:  
- Títulos grandes, en negrita, color #1F2933.  
- Subtítulos medianos, peso semibold.  
- Textos normales en gris azulado.  
- Botones con texto claro y fuerte.  
  
### Componentes visuales  
  
Los botones principales deben ser azules, con esquinas redondeadas y buen padding.  
  
Ejemplo:  
- Fondo: #007BFF  
- Texto: #FFFFFF  
- Hover: #006BE6  
- Border radius: 14px  
  
Los botones secundarios deben tener fondo blanco, borde azul claro y texto azul.  
  
Las tarjetas deben tener:  
- Fondo blanco.  
- Border radius entre 18px y 24px.  
- Sombra suave.  
- Bordes muy sutiles en azul claro.  
  
### Layout  
  
Usar un diseño dashboard moderno:  
- Sidebar lateral limpia.  
- Barra superior con buscador, usuario y notificaciones.  
- Área principal con tarjetas de información.  
- Módulos organizados en grid.  
- Espacios amplios entre secciones.  
  
### Estilo de iconos  
  
Usar iconos lineales, redondeados y simples, preferiblemente con trazos azules o gris oscuro.  
  
### Detalles decorativos  
  
Agregar pequeños elementos gráficos inspirados en la imagen:  
- Puntos azules agrupados.  
- Círculos concéntricos suaves.  
- Líneas curvas azules.  
- Rectángulos flotantes azul claro.  
- Pequeños acentos verdes y naranjas.  
- Flechas simples tipo dibujo manual.  
  
Estos elementos deben usarse como decoración secundaria, nunca deben sobrecargar la interfaz.  
  
### Sensación de marca  
  
La aplicación debe sentirse:  
- Moderna  
- Digital  
- Ordenada  
- Confiable  
- Amigable  
- Productiva  
- Ligera
